# petemcw.github.io

It's my website.

## Running locally

```bash
git clone git@github.com:petemcw/petemcw.github.com.git
cd petemcw.github.com/
bundle install
bundle exec jekyll serve --watch
```

## Publish a new version

```bash
rake publish
```
